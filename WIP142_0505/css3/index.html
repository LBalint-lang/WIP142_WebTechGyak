Ez az oldal fejlece

A CSS (angolul cascading style sheets) a számítástechnikában egy stílusleíró nyelv, mely a HTML vagy XHTML típusú struktúrált dokumentumok megjelenését irja le.


A CSS elnevezésénél a Cascade szó rangsorolásra utal, tehát nem mindegy, hogy az adott stílus hol helyezkedik el a rangsorban. Az ütközések elkerülése végett még pár egyszerű szabály létezik. Az öröklés az XHTML elemek egymásba ágyazottságán alapul, így az egymásba ágyazott elemek öröklik az őket tartalmazó elem beállításait. A szűkítésnek a stilusok súlyozásánál van szerepe. Azaz ha meg akarjuk különböztetni egy stílus fontosságát, akkor szűkebben, vagyis pontosabban kell megadnunk azt, így vehetjük rá a böngészőt, hogy használja. A stílusok elhelyezkedésüktől függően fontossági sorrendben állnak: Böngészőstílus, Felhasználói stílus, Beágyazott stílus, szövegközi stílus.
